Dalvis
This is a web app for connecting gadget technicians and their costumers. The costumers simply specifize their gadget and its breakage, and their gadget will be taken to the nearest, available technicians. Their gadget will be returned back to their home after the repairment is finished.

My role on this project is on both front-end and back-end.
The technology i used for this project are:
HTML, CSS, JS (with React JS), JSON, AJAX, PHP (with Laravel), and MySQL. I also used Google Map API for easing detecting the costumers and technicians location.
NOTE: Above link are only demo i've built with: HTML, CSS, JS, and JQuery.
